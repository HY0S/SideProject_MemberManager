# SideProject_MemberManager

홀덤펍 현장에서 고객 점수와 빙고 이벤트를 관리하기 위한 유저 관리 프로그램입니다.

## 문서

- **[프로그램 기획서](docs/프로그램_기획서.md)** — 요구사항, 기능, 화면 구성, 데이터 설계, 개발 일정 등

## 기술 스택 (예정)

- **언어**: Java 17+
- **UI**: Swing 또는 JavaFX
- **저장**: JSON 파일 + 자동 백업
- **대상 환경**: Windows 10 이상

## 주요 기능

- 고객(닉네임) 등록·삭제
- 행동 기반 점수 추가·관리
- 가변 크기(2×2 ~ 6×6) 빙고판 관리 및 빙고/올빙고 판정
- 관리자 비밀번호 접근 제어
- 데이터 무결성 보장(분리 저장, 백업)
